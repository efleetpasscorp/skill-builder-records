# Skill-Builder Contemporaneous Records

Record-keeping repository for Leonardo's daily skill-builder curriculum, covering two tracks: **R&D Management** and **Contemporaneous Records**. Structured as a Systematic Progression of Work (SPW) so the trail is defensible under the same contemporaneous-documentation logic as ITAA 1997 s 355-25 (R&D core-activity continuity).

## Structure

```
skill/
├── README.md                                  ← this file
└── references/
    ├── learning-log.md                        ← daily SPW entries (one per session)
    ├── run-sheets/
    │   ├── INDEX.md                            ← run-sheet index + integrity hashes
    │   ├── RUN-2026-06-12-01.md
    │   ├── RUN-2026-06-13-01.md
    │   └── RUN-2026-06-14-01.md
    └── weekly-reports/
        └── 2026-06-14-weekly-report.md         ← Friday weekly synthesis
```

## Conventions
- **Daily entries:** `## YYYY-MM-DD — topic` appended to `learning-log.md` (never overwritten).
- **Run sheets:** `RUN-YYYY-MM-DD-NN.md`, one per session; carry the evidentiary detail.
- **Weekly reports:** `YYYY-MM-DD-weekly-report.md`, dated to the week-ending day.
- **Integrity:** content hashes in `run-sheets/INDEX.md`; git commit history is the authoritative immutable audit trail.

## Provenance of this first compilation
Files were compiled 2026-06-15 from verified session transcripts of three completed runs (Session IDs recorded inside each run sheet). The 2026-06-12 run date is inferred from the daily schedule and flagged; 2026-06-13 and 2026-06-14 are confirmed. Retrospective compilation is disclosed rather than concealed. Future sessions should file each run sheet on its own day so the chain is fully contemporaneous.
